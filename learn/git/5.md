# Day 5 - Branching

## Summary

- Branches let you work on changes separately.
- `git branch` shows or creates branches.
- `git switch` changes the active branch.
- You can merge a branch back after work is ready.

## Commands

- `git branch` → Lists or creates branches.
- `git switch` → Moves to a different branch.
- `git log --graph` → Shows branch history in a graph.

## Notes

- Keep branch names clear and meaningful.
- Switch branches before you start new work.
- Branches are safe places to try ideas.
