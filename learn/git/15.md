# Day 15 - Stashing Changes

## Summary

- `git stash` saves uncommitted work temporarily.
- `git stash pop` restores the latest stash and removes it.
- `git stash apply` restores work without deleting the stash.
- Stashing helps when you need to switch tasks quickly.

## Commands

- `git stash` → Saves local changes to a stash.
- `git stash pop` → Restores and removes the latest stash.
- `git stash apply` → Restores a stash without removing it.

## Notes

- Use `git stash -u` to include untracked files.
- Run `git stash list` to see saved stashes.
- Stashes are only temporary storage, not a long-term branch.
