# Day 8 - Connecting to a Remote

## Summary

- A remote repository is a hosted copy of your project.
- `git remote add` links your local repo to the remote.
- `origin` is the usual name for the main remote.
- `git remote -v` checks the configured remote URLs.

## Commands

- `git remote add <name> <url>` → Adds a remote connection.
- `git remote -v` → Shows all configured remote repositories.

## Notes

- A remote lets you push and pull code.
- You can rename or add more remotes later.
- Always verify the remote URL before pushing.
